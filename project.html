ProjectQuest

This is school project for a Web Development module. The project required us to identify an existinng problem and develop a web-based solution to solve it. My team identified the following problem: low work productivity during remote work arrangement especially for Software Developers. Thus, to improve the motivation amongst the employees, we created a gamified Software Project Management tool for easy remote team and task management.

The application is built on Bootstrap, JavaScript and Vue.js for the frontend which features animations and guided tours for our pages.
